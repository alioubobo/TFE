# Aller à la Rencontre de Soi

ARS est un site de coaching en ligne, dans le domaine du developpemnt personnel

## Environnement de développement

### Prérequis

PHP 7.3
Composer
Symfony CLI

### Lancer l'environnement de developpement

### commandes

composer install
symfony server -d

### Lancer des tests

### commande

php bin/phpunit --testdox
